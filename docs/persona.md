# MBTI ペルソナ仕様

## 目的

AI と隠れ人間の発言にトーン差を持たせ、単調な機械応答に見えないようにする。同時に **過剰には固定しない** (overfit させない) ことで「型推し当て」が攻略法にならないようにする。

## 基本方針 (元 PR から引き継ぎ)

- 各参加者に MBTI 4 軸の傾向を割り当てる
- それぞれが文体 / 語彙 / 絵文字使用などをわずかに傾ける
- 「○○ 型は必ず○○と言う」のような 1:1 対応は作らない

## 傾けるべき軸

| 軸 | 表現の傾け先 |
|---|---|
| E / I | 発話頻度 / 自発的な追加発言の有無 / スキップ率 |
| N / S | 抽象 ↔ 具体 (比喩を使うか、観察可能な事実を述べるか) |
| T / F | 論点重視 ↔ 共感重視 |
| J / P | 結論先 ↔ 余地を残す / 言い換える |

## 未決事項

- AI と隠れ人間で MBTI 分布を変えるか (ゲームバランス)
- ペルソナ実装の方式 (口調テンプレ vs 自由生成 vs 混在)
- バイアスの強度パラメータ (「弱く効く」のレベル制御)
- 「ペルソナ崩れ」検知 (隠れ人間が一貫性ない場合に得点減?)
