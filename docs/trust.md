# 信頼度モデル

## 目的

隠れ人間 → プレイヤー方向の「信頼度」を持たせ、終局判定で「相手があなたを人間だと見抜いたか」を表現する。

## 仕様 (元 PR から)

- 各隠れ人間がプレイヤーに対して `trustTowardPlayer` を持つ
- 会話中の発言の質 (お題接続度 / 自然さ / 反応) で増減
- 終局時、`TRUST_RECOGNITION_THRESHOLD` 以上なら「あなたを人間と認識した」とみなす
- `humansWhoFoundYou.length >= 1` で `humanSuccess = true`

## 終局判定マトリクス

| プレイヤーが隠れ人間を当てた | 隠れ人間がプレイヤーを認識した | 結果 |
|---|---|---|
| ○ | ○ | 完全勝利 |
| ○ | × | 片側勝利 (プレイヤー側) |
| × | ○ | 片側勝利 (隠れ人間側) |
| × | × | 完全敗北 |

## 未決事項

- `TRUST_RECOGNITION_THRESHOLD` の値と決め方 (固定値? お題依存?)
- 増減ルールの重みづけ (どの行動が何ポイント動かすか)
- プレイヤー → 隠れ人間方向の「信頼度」も持たせるか (現状は片方向)
- 隠れ人間が複数いる場合の合成 (全員過半 / 1 人でも閾値超え / 平均, など)
- 終局前に「あの人怪しい」フラグを立てる中間 UI を出すか
