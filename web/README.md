# 人間を当てろゲーム

AIに紛れた人間を会話で見抜きつつ、相手の人間にも自分を見抜いてもらう iPhone 向けのブラウザゲームです。

## できること

- AI人数 2〜12、人間人数 2〜4 を可変にしたルーム設定
- お題付きの会話ラウンド
- MBTI の軽いバイアスを持つ AI / 隠れ人間の発言
- 各ラウンドの打ち込み猶予と、1ラウンド1発言の連投制限
- お題キーワードがない発言を禁止する簡易オフトピック制御
- 完全勝利 / 片側勝利 / 完全敗北の3段階判定

## 起動方法

ビルドは不要です。静的ファイルをそのまま配信してください。

```bash
cd <project-directory>
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000/index.html` を開くと遊べます。iPhone では Safari で開けばそのまま動作します。

## 補足

- この実装はローカルで遊べる MVP です。
- プレイヤー以外の人間は「隠れ人間」としてシミュレーションされています。
- 将来的に実際の複数人対戦に拡張する場合は、ルーム管理とリアルタイム通信を追加してください。
